If the system setting means 'user prefers reduced motion', you will see a red background box and two green background box below, otherwise you will see a green box and two red box.
You can find the setting for prefers-reduced-motion
  • On Windows 10; Ease of Access -> Other options -> Play animations in Windows
  • On Linux in gnome-tweaks; Appearance -> Animations
  • On MacOSX; System Preferences > Accessibility > Display > Reduce Motion
    • @media (prefers-reduced-motion: no-preference)
    @media (prefers-reduced-motion: reduce)
    @media (prefers-reduced-motion)
    NOTE: On MacOSX the system setting change doesn't reflect until restart Firefox or open the content in a new window due to bug 1486971
    NOTE2: On all platforms the system setting change doesn't reflect the content in sub frames, for example, if you open this content in an iframe, you will not see the background color changes until you reload the content. (bug 1478212)
